Frequently Asked Questions

常见问题

Important things to know before using CC Switch

使用 CC Switch 前需要了解的重要事项

Why does the app request folder access permissions?

为什么应用需要请求文件夹访问权限？

+

CC Switch needs to read and modify your shell configuration files (.zshrc, .bashrc, etc.) to update environment variables. When you first run the app, macOS will prompt you to grant access to your home directory or specific folders. This permission is required for the app to function properly. All file operations are performed locally on your device.

CC Switch 需要读取和修改您的 shell 配置文件（.zshrc、.bashrc 等）来更新环境变量。 首次运行应用时，macOS 会提示您授予对主目录或特定文件夹的访问权限。 此权限是应用正常运行所必需的。所有文件操作都在您的设备上本地执行。

Why don't configuration changes take effect immediately?

为什么配置更改不会立即生效？

+

When CC Switch updates your shell configuration file, the changes only affect new terminal sessions. Existing terminal windows will continue using the old environment variables. To apply changes to current terminals, you need to manually run:

当 CC Switch 更新您的 shell 配置文件时，更改只会影响新的终端会话。 现有的终端窗口将继续使用旧的环境变量。要将更改应用到当前终端，您需要手动运行：

source ~/.zshrc # or your custom config file # 或您的自定义配置文件

Tip: You can also close and reopen your terminal to load the new configuration.

提示：您也可以关闭并重新打开终端来加载新配置。

How do I set up a custom shell configuration file?

如何设置自定义的 shell 配置文件？

+

By default, CC Switch modifies ~/.zshrc. If you use a different shell or custom configuration file:

默认情况下，CC Switch 会修改 ~/.zshrc。如果您使用不同的 shell 或自定义配置文件：

  1. Open CC Switch Settings (⌘,)
  2. Enable "Use custom config file"
  3. Select your config file (e.g., ~/.bashrc, ~/.profile)
  4. The app will validate the file path and permissions
  1. 打开 CC Switch 设置（⌘,）
  2. 启用"使用自定义配置文件"
  3. 选择您的配置文件（如 ~/.bashrc, ~/.profile）
  4. 应用会验证文件路径和权限

How can I verify that a configuration is working?

如何验证配置是否正常工作？

+

After switching configurations, you can verify the environment variables in a new terminal:

切换配置后，您可以在新终端中验证环境变量：

echo $ANTHROPIC_API_KEY
echo $ANTHROPIC_BASE_URL
claude-code --version

You can also test the connection by running a simple Claude Code command to ensure it's using the correct API endpoint.

您也可以通过运行简单的 Claude Code 命令来测试连接，确保它使用了正确的 API 端点。

What should I do if the app stops working after a macOS update?

如果 macOS 更新后应用停止工作怎么办？

+

macOS updates sometimes reset app permissions. If CC Switch stops working:

macOS 更新有时会重置应用权限。如果 CC Switch 停止工作：

  1. Go to System Preferences → Security & Privacy → Privacy
  2. Select "Files and Folders" or "Full Disk Access"
  3. Ensure CC Switch has the necessary permissions
  4. Restart the app if needed
  1. 前往系统偏好设置 → 安全性与隐私 → 隐私
  2. 选择"文件和文件夹"或"完全磁盘访问权限"
  3. 确保 CC Switch 具有必要的权限
  4. 如有需要请重启应用
